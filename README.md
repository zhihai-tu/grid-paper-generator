# 田字格生成器

一个专为中文教学设计的田字格字帖生成工具，帮助学生根据拼音进行默写练习。

## 🎯 项目简介

田字格生成器是一个在线工具，教师或家长可以输入中文内容，系统自动生成带拼音的空白田字格，供学生根据拼音默写汉字，非常适合小学生的汉字书写练习。

## ✨ 已完成功能

### 核心功能
- ✅ **中文输入**：支持输入任意中文文本
- ✅ **智能拼音转换**：使用 pinyin-pro 库自动生成标准拼音（带声调）
  - 支持整句分析，准确识别轻声（如：个子 → gè zi）
  - 根据上下文智能处理多音字（如：银行 → yín háng）
- ✅ **标准田字格生成**：
  - 每行固定 11 个字
  - 田字格尺寸：60px × 60px
  - 外框：黑色实线（2px）
  - 十字线：灰色虚线（中灰色，短间距）
  - 田字格上方显示拼音（17px 字号）
  - 田字格内部为空白，供学生默写
- ✅ **空格处理**：输入空格时对应位置留空，不显示田字格
- ✅ **换行支持**：支持文本中的换行符强制换行
- ✅ **A4 纸张预览**：预览区域按照标准 A4 纸张尺寸（794px × 1123px）显示
- ✅ **一键打印**：点击生成后自动打开打印预览
- ✅ **打印优化**：专门优化的 A4 打印样式

### 用户体验

- ✅ 清晰的使用说明

## 🚀 使用方法

1. **输入中文**：在文本框中输入要练习的中文内容（支持空格和换行）
2. **生成字帖**：点击"生成字帖"按钮
3. **预览效果**：页面自动滚动到 A4 预览区域
4. **打印使用**：系统自动打开打印对话框，可直接打印

## 🎨 技术特点

### 前端技术
- **HTML5**：语义化结构
- **CSS3**：现代化样式和渐变效果
- **原生 JavaScript**：无需额外框架
- **pinyin-pro v3.24.2**：准确的拼音转换库（通过 CDN 引入）

### 田字格设计
- 标准 60x60px 田字格
- 2px 黑色实线外边框
- 灰色虚线十字（使用 CSS linear-gradient 实现）
- 拼音显示区域 26px 高度，17px 字号
- 每行固定 11 个字，字间距 6px

### A4 纸张规格
- **屏幕预览**：794px × 1123px
- **打印尺寸**：210mm × 297mm
- **打印边距**：15mm 10mm
- **完美适配**：所见即所得

### 拼音智能处理
- **整句分析**：使用 `type: 'array'` 模式进行整句转换
- **轻声识别**：自动识别常见轻声词（个子、妈妈、桌子等）
- **多音字处理**：根据词语上下文选择正确读音
- **准确率高**：相比逐字转换，准确率显著提升

### 打印优化
- 自动隐藏输入界面
- 优化的打印边距
- 良好的分页控制
- 黑白打印友好
- 标准 A4 纸张适配
